ww_metashapelib

A Python library for reading and analysing MetaShape reference and camera calibration files.

read_metashape_reference_file_into_dataframe

 read_metashape_reference_file_into_dataframe (filename)

Read a Metashape reference file into a MetashapeReference class and save the data in a Pandas dataframe. Also save the filename, the metadata from the comment above the header line, the total_error as a Pandas Dataframe, and the contents of the specified file in string, in the returned data structure.

Type Details
filename Metashape reference filename to read.
Returns DataFrame MetashapeReference class data structure.

Generate a list of test reference files.

MetaShape_ref_total_errors

 MetaShape_ref_total_errors ()

This class is populated with the Total Error values from the Agisoft MetaShape reference export file. It is populated by the read_metashape_reference_dir_total_errors() function. Once populated it will containt the path name self.path of the directory where the reference files are located, and a Pandas Dataframe self.df which contains the actual data from the Total Errors.

read_metashape_reference_dir_total_errors

 read_metashape_reference_dir_total_errors (dir_path:str,
                                            mask:str='*-ref.txt')

Reads all MetaShape files matching mask in the specified path. The Total Error from each file is extracted and loaded into a row in a Pandas Dataframe which is returned to the user for analysis.

Type Default Details
dir_path str Path to MetaShape reference data files.
mask str *-ref.txt Mask to select the reference files.
Returns object Class with path and dataframe attributes

 read_metashape_camera_cal_file (filename, pixel_size=5.5e-06,
                                 units='pixels', rv='dict', debug=False)

Read a Metashape xml camera calibration file into a dictionary. Values can be converted from pixels to meters, or millimeters. Be sure to specify the pixel size in meters.

Type Default Details
filename The MetaShape xml calibration file to read.
pixel_size float 5.5e-06 Pixel size in meters.
units str pixels Can be “pixels”, “m”, or “mm”
rv str dict R value. Can be “dict” or “df” or “class”
debug bool False
Returns dict

metashape_cal_to_tsai

 metashape_cal_to_tsai (df, save=True, tsai_file='', path='.',
                        return_str=False, debug=False)

Convert a MetaShape camera calibration Pandas Dataframe to a tsai file.

Type Default Details
df MetaShape cal dataframe.
save bool True True to save tasi to a file.
tsai_file str An optional filename to write the tsai data too.
path str . Path to save the file to.
return_str bool False True if you want the tsai string returned.
debug bool False True debug tis module.
Returns str Returns the entire tsai cal string.
